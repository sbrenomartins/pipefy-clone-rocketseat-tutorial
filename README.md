# Pipefy Clone 

> Clone do Pipefy, utilizando ReactJS e drag n' drop, criado seguindo o tutorial da Rocketseat, feito pelo Diego Fernandes: [https://youtu.be/awRtgpRsdTQ](https://youtu.be/awRtgpRsdTQ)

### Features utilizadas
* "immer"
* "react-dnd"
* "react-dnd-html5-backend"
* "react-icons"
* "styled-components"
